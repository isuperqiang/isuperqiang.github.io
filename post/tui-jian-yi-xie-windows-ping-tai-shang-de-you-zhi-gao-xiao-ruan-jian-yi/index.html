推荐一些 Windows 平台上的优质高效软件（一）

2017-03-04

在 PC 领域苹果电脑是高端的代名词，凭借 MacOS 的优秀设计，Mac 已经成为不少办公族的标配。技术圈里大多数人推荐使用 Mac 开发，但是考虑到成本问题，有的公司还是没能满足开发者的诉求。我个人使用 Windows 系统有几年时间了，对其特性和使用技巧有所了解，这篇文章主要介绍我平时用到的优质的软件，只为提高工作效率。

1. 搜狗输入法

搜狗拼音输入法 是国内用户量最大的输入法，鉴于搜狗这么多年来在输入法领域的精耕细作，其产品的质量还是不错的。自定义短语是非常好用的功能，在属性设置 → 高级设置里面，把使用频度高的短语词汇（比如开发经常用到的 ADB 命令 ）添加进去，使用的时候输入两三个字母就行了，确实非常地方便。

2. 每日必应

桌面是使用电脑的过程中看到最多的界面了，为了让减轻审美疲劳，我们总会经常性地更换壁纸。市面上有不「桌面壁纸」的软件，虽说是免费的，但充斥着各种广告，对用户的干扰太多。我用的是「每日必应」，需要从 Windows 应用商店里下载，安装启动后开启自动更新，同时在 Windows 设置 → 隐私→ 后台应用里面把每日必应开启就行了，这样每天开机都会看到最新的壁纸，图片的质量挺高，支持保存到本地相册，非常地 nice！

3. Clover 资源管理器

众所周知，Windows 上的资源管理器不支持多标签页，也就是说你每打开一个文件夹就得打开一个窗口。如果进行大批量的文件操作，就得打开许多的窗口，工作量可想而知！微软的工程师没有解决这个用户痛点，到现在的 Windows 10 上依然存在问题。Clover （英文的意思是三叶草）就是为解决多标签页的问题而生，它还支持添加书签栏，表面上的使用体验和 Chrome 差不多，但是完胜自带的资源管理器。根据官网的介绍，这应该是一家小公司的产品，但是能做到这样已经算是不错了。

4. Cmder 命令行工具

我想 Windows 被开发者吐槽最多的就是命令行了，界面丑陋、功能太弱、还不支持复制粘贴，扩展性太差。于是乎更好的替代品就出现了，它就是 Cmder，一个集成多种终端、界面美观、功能强大、满足你需求的工具。具体配置可以参考这篇文章 Win下必备神器之Cmder。

5. Ditto 剪贴板管理器

Windows 上没有剪贴板管理器，我们每次使用剪贴板的内容后下次使用还要重新剪贴，这就是重复的劳动。 Ditto 会把我们剪贴的记录保存在数据库，下次使用的时候直接从记录里面查找就行，支持在当前的上下文窗口弹出界面。我喜欢把弹窗的快捷键设置成 Alt + ·（Esc 下面的反斜杠），这样每次查找记录的时候会很方便。官网的介绍是英文，关于使用帮助，大家可以参考这篇文章 Ditto:首选的剪贴板增强软件。

6. f.lux 调节色温 保护眼睛

上班族的白领们每天面对电脑的时间超过 8 小时，对眼睛的刺激和伤害是不容忽视的。我就有这样的感觉，每日天早上的时候眼睛明亮有神，工作效率是极高的，但是到了晚上，就会明显地感觉到眼睛干涩疲劳，工作效率随之下降。f.lux 是我发现十分好用的工具，它根据一天的时间变化调节色温，白天冷色调，夜晚暖色调。更多介绍请参考 f.lux - 全天候保护眼睛健康软件！自动调整屏幕色温减少蓝光防疲劳，长时间玩电脑必备！

7. Foxmail 邮件客户端

或许你还缺一款像样的邮件客户端，尽管 Windows 10 平台上有 Outlook，但是我更喜欢充满本土风格的 Foxmail。同时我选择了 foxmail 作为邮件的主机服务器，这源于它的创作者 —— 张小龙，在做出微信这款产品前他就是靠 Foxmail 被腾讯收入旗下。选择 Foxmail 就是为了向张小龙致敬，一位伟大的产品经理，虽然大家前身都是程序猿。其实最好用的邮件服务提供商是 Google，但是 Gmail 在国内会有使用障碍。最喜欢的就是微信的邮件提醒服务，Foxmail 可以看作是 QQ 邮箱的别名，但是你要是在简历上填写 QQ 邮箱，给人一种非正式的感觉。推荐把 QQ 邮箱和 Foxmail 关联，工作生活两不误。

8. 印象笔记

「大象永远不会忘记」，这是印象笔记 logo 的由来。好记性不如烂笔头，每当我们遇到问题，当时的理解明白是不够的，人的记忆碎片太多，时间稍纵就会遗忘，这就是做笔记的必要性。勤做笔记，时常温故，总会有新的启发。免费用户最多允许两台终端设备登录，每月最大上传流量 60 M。我觉得免费版就够用了，除非你是大学霸，可以付费申请高级账户。

9. Notepad++

Notepad ++ 绝对是替代 Windows 下默认记事本的利器，显示行号、语法高亮、智能提示、界面性感……总之，放弃系统自带的记事本的吧，你没有理由不爱它。

10. 网易云音乐

在音乐中引入社交元素，体现出互动性，网易云音乐 做得比其他的播放器出众，音乐推荐又是一大亮点，对版权的支持越来越看重，网易旗下难得的好产品。

这次的推荐就这么多，欢迎大家分享使用心得，下次再与大家分享其他好玩实用的工具~
