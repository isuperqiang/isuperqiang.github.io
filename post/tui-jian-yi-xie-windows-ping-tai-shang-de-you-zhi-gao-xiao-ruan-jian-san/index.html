推荐一些 Windows 平台上的优质高效软件（三）

2017-06-12

最近又发现了不少优秀的软件，准备介绍给大家。前两期的推荐请戳这里~

  • 推荐一些 Windows 平台上的优质高效软件（一）
  • 推荐一些 Windows 平台上的优质高效软件（二）

21. Seer 空格预览

Seer 是一款 Windows 下免费的模仿 macOS 系统文件预览的工具，可以让你不用打开文件就能快速通过按空白键预览文件的内容，这对于电脑中文件较多的用户来说非常实用！Seer 的功能比 macOS 自带的预览功能还丰富，譬如增加了 Markdown 语法显示支持、代码高亮、 JSON 和 XML 格式化查看、文本编码检测、文件夹内容预览等。Seer 的使用及其简单，安装运行后常驻后台，选中任何一个文件，敲一下空格就会出现预览，再敲一下空格关闭预览。

22. 坚果云

坚果云 是一款优秀的云盘服务，相对于国内外的同步网盘都有一定优势，支持多平台、多文件夹同步、选择性同步、误删除恢复 （版本控制）、外链分享、协同办公、局域网加速，而且它还支持增量同步，只上传文件被修改的部分而不是上传整个文件，这样能大大提高同步的效率。免费的坚果云用户不限空间，每月 1GB 的上传流量、3GB 的下载流量，可与五人协作同步。对于轻度使用者来说，空间基本够用。优秀却低调的坚果云，还是值得体验的。

23. EagleGet 下载器

EagleGet 是一款界面清爽、功能强大的网络下载器，采用多线程技术大幅提高下载速度，支持所有主流的浏览器，轻松恢复中断的下载，完成后自动扫描病毒。还可以检测网站中的媒体内容，支持下载网络上的高清视频，这个功能非常实用。最重要的是它是免费的，已经支持简体中文。如果你厌烦了迅雷无节操的广告，EagleGet 会是不错的选择。

24. Avast

Avast 是国外的一款杀毒软件，提供免费版和付费版，使用免费版检测病毒还是够用的，付费版提供了全方位的保护。与国内的各大杀软不同， 国外的杀软非常规矩，不耍流氓，比如广告弹窗、修改主页等，总体来说还是比较省心的。还有一款叫做 Avira（中文名叫小红伞），同样也是不错的，值得推荐。

25. Gaaiho Reader

Gaaiho Reader 是文电通提供的一款免费的 PDF 阅读器，可以媲美福昕阅读器。文电通的产品主要面向企业级用户，出售与 PDF 相关的软件和服务。值得称赞的是，它还提供了免费的阅读器。全屏阅读时指针变成圆点，方便像 PPT 一样做演示。功能齐全，体验良好，没有广告，确实是国产良心软件。

26. WGestures

WGestures 是一款全局鼠标手势软件，绝对是鼠标党的福利！WGestures 里面定义了大量的鼠标操作，包括右键上滑复制、下滑粘贴，还有触发角和摩擦边，支持自定义操作，十分地灵活。有了 WGestures，从此解放键盘上的双手，小小鼠标有大作为~

27. Everything

Everything 是当之无愧的文件搜索神器，它可以在闪电般的瞬间从海量的硬盘中找到你需要的文件，速度不是一般的快。还有它是一款完全免费的软件，界面简洁高效，体积虽然小巧，功能却非常丰富，支持正则表达式搜索。Everything 内置了 HTTP、ETP/FTP 服务器的功能，可以将硬盘变成一个网站或 FTP，方便别人下载你的文件。而且，HTTP 的分享同样拥有一个方便强大的搜索功能。Everything 使用介绍请参考：Everything - 免费极速文件搜索工具神器，绝对快得让你震惊！

28. Snipaste

Snipaste 是一个简洁强大的截图工具，可以让你将截图贴回到屏幕上。打开 Snipaste，按下F1来开始截图，再按F3，截图就桌面置顶显示了。你还可以将剪贴板里的文字或者颜色信息转化为图片窗口，并且将它们进行缩放、旋转、翻转、设为半透明，甚至让鼠标能穿透它们。贴图功能将改变你的工作方式、提升工作效率。Snipaste 是免费软件，它也很安全，没有广告，国产良心软件。截图、标注、贴图和取颜色值是我最常用的功能，不得不说这款软件真的非常棒！

29. Geek uninstaller

Geek uninstaller 是一款小巧好用的软件强制卸载工具，界面非常简洁，不需要安装，下载后直接运行就可以。最主要的是功能强大，利用它卸载软件后，可以深度清理注册表和安装文件夹。虽然说某些电脑管家也能做到，但是不及它专业。除此之外，它还支持搜索安装的软件，做到快速定位。对于 Windows 10 用户来说，还有个额外的功能，就是可以清理 Winsows Store 下载的软件，包块微软预装的全家桶。

30. VMware Workstation

VMware Workstation 是一款功能强大的桌面虚拟计算机软件，允许一台真实的电脑在一个操作系统中同时打开并运行数个操作系统，免费版本为 VMware Workstation Player。完全隔离并且保护不同 OS 的操作环境以及所有安装在 OS 上面的应用软件和资料。不同的 OS 之间还能互动操作，包括网络、周边、文件分享以及复制贴上功能。能够设定并且随时修改操作系统的操作环境，如：内存、磁盘空间、周边设备等等。我用它安装了 macOS 和 Ubuntu 系统，整体感觉还是比较稳定的，有特殊需要或者想体验其他系统的朋友可以试试这款虚拟机。

工作当中，效率为先。希望这些优质软件能为你带来幸福感，欢迎大家留言交流，嘻嘻……
