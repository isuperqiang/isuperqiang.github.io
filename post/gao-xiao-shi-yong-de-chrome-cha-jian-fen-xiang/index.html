高效实用的 Chrome 插件分享

2017-06-24

Google Chrome 是我最喜欢的网页浏览器，极速的浏览体验、丰富的扩展程序和强大的开发者工具，让每个使用 Chrome 的人都爱不释手。今天，我就把自己使用的扩展程序分享给大家，它们将极大地提高我们的工作效率。

下面介绍的插件都可以在 Chrome 网上应用店里面搜索下载 O(∩_∩)O~

1. Adblock Plus

这是世界上最受欢迎的广告拦截软件之一，能够拦截网页上的大部分横幅、弹窗和视频广告，让我们享受没有恼人广告的网络世界，10,000,000+ 位用户的选择，你值得拥有 (o゜▽゜)o☆

2. Chrono 下载管理器

这绝对是 Chrome 中最好的下载管理器，支持按照文件类型分类，查看下载的文件详情，方便管理下载的文件，支持自定义下载任务。「除了高效管理下载任务，Chrono 资源嗅探器可以检测网页上的所有链接、图片、音频和视频，你还可以根据链接的文件类型或正则表达式来筛选你要下载的地址。」，这是官方的介绍，我还没有试过它的嗅探功能，大家可以慢慢探索~

3. Proxy SwitchyOmega

轻松快捷地管理和切换多个代理设置，支持备份和恢复，自定义情景模式，科学上网的时候用得比较多，也是非常好用的工具。

4. 极简图床

当我们写博客的时候，总需要图床工具来存放图片，但是图片上传后不容易管理。极简图床是不错的选择，支持粘贴上传、拖放上传，一键复制 markdown 链接，可自定义七牛账号信息，一键采集网页中所有图片。从使用体验来说，还是非常贴心的~

5. LastPass

平时注册的账号太多，都用同一个密码不安全，用不同的密码又记不住，这时候 LassPass 就能大展手脚了。它是一款免费的密码管理器，能记住各大网站的账户密码，支持多设备之间同步，还可以查看密码明文。当然首先得注册 LassPass 的账号，这样就相当于有了个保险箱，妈妈再也不用当心我忘记密码了~

6. Momentum

Chrome 默认打开新标签页的时候，是默认的 Google 搜索框，还有最近打开的网站，看久了就会觉得无聊。Momentum 是个壁纸插件，每天自动更换新标签页的背景，图像的质量还是比较高的。

7. Imagus

鼠标指针悬停在链接或缩略图上时，直接在当前页面的弹出视图上显示这些图片、HTML5 视频/音频和内容专辑。使用最多的便是 Imagus 的自动加载图片并放大显示功能，省去了点击查看大图的操作，用久了就知道这真的是神器！

8. Click&Clean

当浏览器关闭时，这款应用程序删除你的浏览历史,防止他人跟踪你的网上活动。为了上网不留痕迹，使用Click&Clean 清除隐私数据、缓存和 Cookie 非常容易，支持指定清理的级别，插件的功能不止这些，总之非常强大！

9. Bookmark Manager

Google 出品的书签管理器，界面风格现代化，CTRL + D 快速添加书签，全局搜索书签，支持方便地为书签分类，非常容易进行导入和导出，登录 Google 账户后书签会自动同步。Google 出品，必属精品！

10.印象笔记·剪藏

使用印象笔记扩展程序，一键保存精彩网页内容到印象笔记帐户。看见什么，「剪」什么。支持高亮划出重点做标记，网页智能优化，可选择保存整个页面、网页正文或指定部分。一键保存后多设备同步，不用复制粘贴编辑再整理。总之，我经常用~

今天就先介绍这么多，推荐大家安装使用，记得发表使用体验哦~
