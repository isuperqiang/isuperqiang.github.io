推荐一些 Windows 平台上的优质高效软件（二）

2017-05-09 5 min read # 软件工具

书接上篇推荐一些 Windows 上平台的优质高效软件（一），继续高效优雅的软件探索之路。

11. TIM -- 轻聊版 QQ

「轻聊的 QQ，更方便办公」这是 TIM 的官方宣传语，简约即是美。相比 QQ 而言，它是非常干净简洁的。没有烦人的广告以及各种增值服务，大部分 QQ 上的功能它都有。在线文档是亮点，适合团队协作。腾讯之前推出的「QQ 轻聊版」很久没有更新了，估计已经弃疗，QQ 办公领域的重心会是 TIM。

12. Listary -- Windows 上的 Spotlight

Listary 是一款非常强大的文件浏览增强工具， 可以让文件在你指尖流动。它为 Windows 自带的资源管理器添加了许多实用的功能，包括智能命令、最近文档以及收藏功能。Listary——让文件在指尖流动 这篇文章介绍得非常好，十分推荐大家使用 Listary，体验双手不离键盘的专注。

13. ManicTime -- 时间追踪

早上开机上班，晚上关机下班，不知不觉一天过去了，想想今天都干什么了，貌似记不清了哎。还好有这么一款应用 ManicTime，它可以记录你在电脑上一天的活动，不管你浏览网页还是编辑文档，它都能分秒不差地记录下来。支持备份和导出数据，有助于分析时间的分配和使用，标准版就够用了，感兴趣的朋友可以试试~

14. Bandizip -- 终极压缩

Bandizip 是一款小巧精致的压缩软件，支持绝大多数文件格式，它是免费的，而且没有广告，打开压缩包时支持预览图片。最值得称赞的是自动解压功能：如果压缩包内是单个文件或文件夹，那就直接解压；如果是多个文件或文件夹，就以文件名新建一个文件夹，然后再进行解压。这样就不会出现解压后出现一大堆文件的问题，非常人性化。还有许多特性，详情介绍参考这篇文章 你的最后一款 Windows 解压缩软件：Bandizip

15. XMind -- 思维导图

XMind 是一款头脑风暴和思维导图工具，利用它可以整理思路，记录笔记，分享想法。可以制作树状图、鱼骨图、甘特图等，支持导出图片、PDF 等格式。如果你只是简单作图，免费账户的功能基本够用，高级账户要 500 多人民币呢！不管怎么说，XMind 功能是非常强大的，推荐大家入手使用。参考介绍：思维导图，原来Xmind这么强大！

16. CCleaner

CCleaner 是一款免费的系统优化和隐私保护工具，主要用来清除 Windows 系统产生的垃圾文件，以腾出更多硬盘空间。它的另一大功能是清除使用者的上网记录。CCleaner 的体积小，运行速度极快，可以对临时文件夹、历史记录、回收站等进行垃圾清理，并可对注册表进行垃圾项扫描、清理，附带软件卸载等功能，还是值得一试的。

17. Honeyview

Honeyview 是一款支持多种格式的极速图像浏览器，它和 Bandizip 是由同一家公司出品，体积小巧，渲染速度快，还是免费的软件。经过简单的设置就可以实现无边框看图的效果，是不可多得的图像浏览器和漫画阅览器。

18. Chrome 浏览器

谷歌浏览器 （Chrome），是一款由谷歌公司开发的开放原始码网页浏览器。迄今为止最好用的浏览器，没有之一！强大的扩展程序为 Chrome 提供了无限可能，开源的 chromium 养活了大多数的国产浏览器，可以说没有 chromium，就没有各种极速的浏览器。虽然国内的谷歌搜索不能使用，但是谷歌公司为我们留下的 Chrome 真的是良心之作。

19. PotPlayer 播放器

PotPlayer 是一款内存占用低、无广告、功能强大的本地播放器，拥有强大的内置解码器。之前一直用 QQ 影音，但是官方一年多没给更新，于是寻找替代的播放器。刚刚接触 PotPlayer，更多功能还在探索中，看样子确实很专业啊！

20. Typora -- Markdown 编辑器

Typora 是一款非常出色的 Markdown 编辑器，特点是所见即所得（What You See Is What You Get），遵循极简的设计风格，界面没有多余的元素，让你能够安心写作。插入表格、图片等非常方便，支持预览目录大纲，还有多套主题随心切换，最重要的它是免费的。纵观众多 Markdown 编辑器，无出其右！少数派这篇总结得详细：让 Markdown 写作更简单，免费极简编辑器：Typora
